# **App Name**: Vorgawall Chat

## Core Features:

- Header Branding: Display 'Vorgawall Demo' branding, chat history link, and navigation back to 'vorgawall.shop'.
- Welcome Message: Display a central welcome message in the chat interface.
- Neon Ring Chat Input: Implement a rounded-rounded-3xl chat input with a neon ring using orange, peach, and soft blue gradient colors, including placeholders and dynamic buttons for attachments (Plus), tools (Settings2), and microphone/send.
- Animated Chat Bubbles: Create rounded user-assistant chat bubbles with a bg-neutral-800 text-white color scheme, complete with animation dots and streaming text effects.
- Tools Menu: Implement a settings menu.
- Microphone Input: Allows users to toggle recording audio snippets, sending them once finalized.

## Style Guidelines:

- Primary color:  very dark with cyber feel (#0a0a0a).
- Background color: White milk (#F0F0F0).
- Accent color: bg-neutral-800 text-white to highlight interactive elements.
- Body and headline font: 'Inter', sans-serif, to provide a blend of modernity with slight warmth for both headers and chat text.
- Use simple, outlined icons for the settings and attachment buttons to maintain a clean interface.
- Ensure the chat interface is responsive, adapting to different screen sizes for optimal viewing on various devices.
- Implement smooth transitions for button states and chat bubble appearances to enhance user experience.
