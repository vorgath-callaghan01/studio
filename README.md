# Basic Chatbot by Vorgawall Shop

Welcome to the **Basic Chatbot**, a high-performance, modern AI chat interface developed by Vorgawall Shop. This project is designed as a lightweight yet feature-rich starter for anyone looking to build a professional-grade chatbot experience using the latest web technologies.

## ‚ú® Key Features

- **Modern UI/UX**: Built with a sleek, minimalist aesthetic featuring `rounded-3xl` components and high-quality transitions.
- **Local Storage Persistence**: All chat sessions and messages are saved directly in the user's browser, ensuring privacy and continuity without a backend database.
- **Dynamic Feature Modes**: Includes specific modes for **Search**, **Image Generation**, and **Article Writing**, each with unique placeholder behaviors.
- **Multi-File Support**: Integrated image gallery and document upload functionality with real-time horizontal previews.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.
- **Chat Management**: Rename, delete, and track chat history with ease.
- **Privacy First**: Data never leaves the client's device.

## üìÅ Project Structure

- `src/app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable UI components (ChatInput, ChatHeader, etc.).
- `src/components/ui/`: Radix-based primitive components from Shadcn.
- `src/hooks/`: Custom React hooks for utility functions.
- `src/lib/`: Utility functions and placeholder data.
